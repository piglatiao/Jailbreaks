AI越狱提示词集合
项目概述
本仓库旨在有组织地收集最新的AI越狱提示词，重点关注适用于NSFW（不适合工作场合）内容写作的提示词。这些提示词来自各种公开来源，并非由仓库所有者原创。项目的目标是为开发者、研究人员和爱好者提供一个集中、结构化的资源，以探索AI模型通过越狱技术展现的行为。
注意：这是一个引用项目，所有内容均来自公开渠道，仓库所有者不主张对提示词的所有权或创作权。
项目目的
本项目旨在：

整理和归类特别适用于NSFW内容生成的越狱提示词。
随着新提示词的出现，保持集合的实时更新。
为研究AI模型限制、安全机制和响应行为的人员提供参考。

为什么聚焦NSFW？
某些AI模型在使用特定越狱提示词时，对生成NSFW内容更宽松或更有能力。本仓库专注于这些模型和提示词，以更好地理解它们的性能和局限性。此集合仅用于道德研究和测试目的。
更新说明
本仓库会不定期更新，添加新发现或分享的越狱提示词。由于越狱提示词具有时效性（AI模型经常更新以修复漏洞），提示词的有效性可能随时间变化。请始终检查最新提交的时间戳以确保相关性。
使用方法

浏览提示词：查看/prompts目录，获取分类整理的越狱提示词。
负责任测试：在受控且符合道德规范的环境中使用这些提示词，遵守被测试AI模型的服务条款。
贡献内容：如果你发现了新提示词或有组织建议，欢迎提交拉取请求或创建问题。

目录结构

/prompts：包含以markdown或文本文件形式分类的越狱提示词。
/docs：额外文档，包括提示词有效性和模型兼容性的说明。
/archive：因模型更新可能失效的旧提示词。
